Association rule mining will find the frequent data items that are found together in the data set

Steps: 1: Select confidence and support 2: You can see the frequently together seen data items from the table below

Confidence: Confidence is an indication of how often the rule has been found to be true.
Support: Support is an indication of how frequently the item set appears in the data set.
LHS: (Left hand side) item or item set that is found in the data set.
RHS: (Right hand side) item or item set that is found together with LHS data item or data items
Iterations: How long will the apriori calculate relationships (quite low is often good)
